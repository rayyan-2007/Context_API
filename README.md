# React Context API Counter

A simple React project demonstrating how to use the **Context API** to manage global state without prop drilling. This project implements a counter with increment, decrement, and reset functionality while sharing state across multiple components.

## Features

* Global state management using React Context API
* Increment counter
* Decrement counter
* Reset counter
* Clean and reusable component structure
* Beginner-friendly project
* Built with React and Vite

## Tech Stack

* React
* Context API
* JavaScript (ES6+)
* Vite

## Getting Started

### 1. Clone the repository

```bash
git clone <your-repository-url>
```

### 2. Navigate to the project

```bash
cd <repository-folder>
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Build for production

```bash
npm run build
```

## What You'll Learn

* What React Context API is
* Why Context API is useful
* How to create a Context
* How to provide data using a Provider
* How to consume Context with `useContext`
